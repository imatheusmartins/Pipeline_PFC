Projeto de TCC

Deep learning aplicado ao diagnóstico assistido de retinopatia diabética

Este projeto apresenta uma proposta voltada ao diagnóstico assistido de retinopatia diabética por meio da aplicação de técnicas de deep learning.

Nesta etapa inicial, a página também descreve a estrutura de CI/CD adotada para apoiar a organização, a validação e a publicação das versões do projeto de forma contínua e controlada.

Desenvolvimento As atualizações são realizadas na branch de desenvolvimento para facilitar a evolução do projeto.
Integração contínua O workflow de CI executa verificações automáticas para garantir a consistência mínima da aplicação.
Controle de versão As alterações aprovadas são integradas à branch principal para manter uma versão consolidada do sistema.
Entrega contínua Após a atualização da branch principal, a publicação da página pode ocorrer de forma automatizada.

Estrutura inicial de CI/CD

  1. O desenvolvimento das alterações acontece na branch dev.
  2. A integração contínua verifica automaticamente se os arquivos essenciais do projeto permanecem válidos.
  3. Depois da validação, as mudanças podem ser enviadas por meio de um Pull Request para a branch master.
  4. Com a aprovação e o merge, a etapa de entrega contínua publica a versão mais recente da página.
Projeto estruturado para acompanhamento, validação e publicação contínua de versões.
