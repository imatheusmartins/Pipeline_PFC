Simulacao CI/CD

Deep Learning como ferramente de auxílio a detecção de retinopatia diabética.

Esta pagina foi criada para demonstrar um pipeline CI/CD simples com GitHub Actions e GitHub Pages, simulando a entrega continua de uma aplicacao do projeto de TCC.

O foco do TCC e investigar o uso de modelos de deep learning como ferramenta de auxilio na deteccao de retinopatia diabetica a partir de imagens de fundo de olho.

Branch de desenvolvimento `dev` recebe alteracoes e dispara a etapa de integracao continua.
Branch principal `master` representa a versao pronta para publicacao.
CI Valida se o HTML existe e contem elementos esperados.
CD Publica automaticamente a pagina quando houver push na `master`.
Pipeline de exemplo pronto para apresentacao academica.
